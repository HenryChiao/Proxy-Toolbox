# 🪟 Windows - Mihomo Clients

## 🟢 一类 (推荐)

### FlClash
* **优势**: 异常勤快的更新频率、轻量化，对 Meta 写法的配置文件可视化适配极佳。
* **缺点**: UI 风格过于“手机化”。

### Clash Verge Rev (CVR)
* **优势**: 目前的爆款。高度覆写性，集成轻量模式/内存占用少，更新积极。
* **缺点**: UI 一般，≥2.3.0 版本 Bug 渐多。

### Sparkle (原 Mihomo Party 分支 A)
* **背景**: 开发者是 Mihomo 核心成员之一。
* **优势**: 对 Meta 写法适配极佳。
* **缺点**: 过于臃肿，随缘更新。

## 🟡 二类

### Clash Party (原 Mihomo Party 分支 B)
* **说明**: 已切换 OpenClash 作者的 Mihomo 分支。
* **风险**: 项目易主，继任者为利益相关者。

### GUI.for.Clash
* **特点**: 模块化的插件中心，点鼠标生成配置。
