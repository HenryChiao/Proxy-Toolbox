# 🤖 Android - Mihomo Clients

## 🟢 一类 (推荐)

### FlClash
* **特点**: 仿 Surfboard UI，但功能更强。
* **优势**: 全协议支持、分流、相对省电、异常勤快的更新频率。对 Meta 写法的配置文件可视化适配极佳。
* **缺点**: 覆写可选项比 CMFA 少。

### Clash Meta for Android (CMFA)
* **特点**: MetaCubeX / Mihomo 官方分支。
* **优势**: 全协议支持、分流、更新积极、覆写可选项全。
* **缺点**: UI 一般，交互逻辑不太合乎直觉。

## 🟡 二类

### Clash Mi
* **核心**: 基于 Mihomo 核心。
* **优势**: 全协议支持、分流。
* **来源**: KaringX 的新开项目。

### YumeBox 🆕
* **核心**: 基于 Vernesong/mihomo 分支核心。
* **优势**: UI 简洁美观；集成 sub-store，支持 smart 策略组。
* **缺点**: 策略组仅支持横排显示，切换繁琐。

### FlyClash
* **说明**: Mihomo 类的另一 GUI 客户端，界面美观。
* **状态**: 较新的项目，代码质量遭非议。

## 🟠 另类 (依赖 Root)

### Surfing 🆕
* **特点**: 从成熟项目 Box4Magisk 分支而来。
* **优势**: 相对其他框架类工具，集成了开箱即用的配置文件。
* **依赖**: Magisk / KernelSU / APatch。

### Box for Root (BFR) / Box4Magisk
* **特点**: 一站式包揽 mihomo, sing-box, v2ray, xray 等全明星阵容核心库。
* **模式**: 支持 redir / TProxy / TUN 等多模式。

### akashaProxy 🆕
* **核心**: 基于 mihomo，支持 TProxy/TUN 模式。

### Clash MIX
* **核心**: 基于 mihomo，支持 TUN 模式。
