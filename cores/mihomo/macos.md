# 🍏 macOS - Mihomo Clients

## 🟢 一类 (推荐)

### FlClash
* **优势**: 支持在状态栏切换节点，对窗口管理低效的 macOS 较便捷。

### Sparkle
* **优势**: 支持状态栏切换节点。

### Clash Verge Rev (CVR)
* **缺点**: 与 macOS 状态栏的融合做得不如 FlClash / Sparkle。

## 🟡 二类

### Clash Party
* **评价**: 在 macOS 比在 Win 上更丝滑，但存在所有权风险。

### ClashMac 🆕
* **特点**: 基于 mihomo 核心，利用 SwiftUI 开发的闭源代理工具。
* **评价**: 美观且轻量，产品逻辑类似 ClashX Meta。

### ClashX Meta
* **来源**: 对原 ClashX 的 fork，由 MetaCubeX 官方维护。
* **缺点**: UI 形如 Meta XD 面板打包，配色诡异；易与其他 App 冲突。
