# 🐧 Linux - Sing-box Clients

## 🟢 一类

### Singbox for Linux
* **特点**: 官方 GUI。
* **优势**: 全协议支持、最强 DNS 处理、衔接 Tailscale。
* **缺点**: GUI 简陋，配置难。

## 🟡 二类

### GUI.for.SingBox
* **说明**: 模块化插件中心，易用性较高。

## 🟠 三类

### Pantheon 🔴 (私有化)
* **说明**: Web 封装 GUI。

### OneBox 🆕
* **说明**: UI 简洁。

### Nekobox for PC 🆕
* **说明**: 对原 Nekoray 的 fork。

## ⚓️ 另类 (软路由/透明代理)

### Sing-box [裸核]
* **部署**: 利用 `nftables` 以 `auto_redirect` TUN 模式运行。
* **评价**: 部署简便、性能极强。
