# 🍏 macOS - Sing-box Clients

## 🟢 一类

### Singbox for Mac (sfm)
* **特点**: 官方客户端。
* **优势**: 全协议支持、最强 DNS 处理、衔接 Tailscale 内网穿透。
* **缺点**: GUI 简陋，学习曲线陡峭。

## 🟡 二类

### GUI.for.SingBox
* **特点**: 易用化的 Singbox，功能同 Windows 版。
* **注意**: > v1.8.9 版因无签名，须借助 `xattr -cr` 或 macOS 小助手绕签。

## 🟠 三类

### OneBox 🆕
* **特点**: 基于 singbox 核心，UI 简洁。
