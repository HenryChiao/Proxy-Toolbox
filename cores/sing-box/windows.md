# 🪟 Windows - Sing-box Clients

## 🟢 一类

### V2rayN (切换核心)
* **说明**: V2rayN 可通过切换核心 (`Sing-box` core) 来支持相关协议。
* **优势**: 稳定、老牌、协议覆盖率第一。

## 🟡 二类

### GUI.for.SingBox
* **特点**: 易用化的 Singbox。
* **功能**: 点鼠标即可生成配置文件，模块化的插件中心（可拓展 SpeedTest、规则集、云盘签到、OpenList、Substore 等 40+ 种）。
* **缺点**: 配置文件只能靠点鼠标生成，无法被手搓/微调的外部 json 覆盖。年轻插件待时间检验。

### Karing
* **说明**: 参考 Android 版，全平台通用。

## 🟠 三类 / 其他

### Pantheon 🔴 (私有化)
* **形式**: 基于 sing-box 核心，以 Web 封装的 GUI 客户端。
* **状态**: 项目已改为私有。

### Sing-Box Windows
* **形式**: 易用化的 Singbox，可自定义配置文件运行。
* **缺点**: 截至 v1.7.2 仍不完善（路径设置 Bug、策略组显示问题）。

### OneBox 🆕
* **特点**: 基于 singbox 核心，UI 简洁。
* **状态**: 较新的项目。

### lvory (Ivory)
* **特点**: 简单无覆写。
* **缺点**: 窗口管理按钮过于 macOS 化，视觉割裂。

### Nekobox for PC 🆕
* **定位**: 自建党 / 单节点梭哈。
* **说明**: 对原 Nekoray 的 fork。

### Nekoray 🔴 (已断更)
* **说明**: UI 优化的 v2rayN，调用 Singbox/v2ray 核心。
