# 🤖 Android - Sing-box Clients

## 🟢 一类 (推荐)

### Sing-box for Android
* **特点**: 官方客户端。
* **优势**: 全协议支持、分流、最强 DNS 处理、省电优化好、衔接 Tailscale 内网穿透、更新积极、免费。基本能做到一个配置文件通杀《全平台》。
* **缺点**: GUI 极其简陋、学习曲线陡峭；配置难度高/覆写全靠手搓 `config.json`；迭代激进/旧配置文件可能跨版本不兼容。

## 🟡 二类 (易用性改良)

### Husi (虎兕)
* **定位**: ～自建党 / 单节点梭哈
* **核心**: 基于 sing-box 核心。
* **优势**: 稳定、简单易用、链式代理易操作。新协议、新特性迅速跟进，协议覆盖率属前列。提供 Rule-set 匹配查询功能。
* **缺点**: 分流繁琐；UI 走经典的 Material Design 公模风。

### Karing
* **核心**: 基于 sing-box 核心。
* **优势**: 分流规则开箱即用、对新手最友好的 sing-box GUI 之一，全平台通用。
* **缺点**: 覆写可选项少，最大缺点可能是 UI 一般。
