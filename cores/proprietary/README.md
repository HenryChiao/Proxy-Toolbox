# 📦 闭源/商业核心 (Proprietary / iOS Giants)

> **简介**: 这些软件不使用通用的开源内核（或核心代码闭源），通常由商业公司或独立开发者维护。它们主要集中在 **iOS/macOS** 生态，且价格较高，但提供了最顶级的 UI 交互和稳定性。

## 🍎 iOS / macOS (三巨头 + 新秀)

### Surge (🇺🇸$49.99 / $99.99)
* **地位**: 👑 代理工具界的“爱马仕”，最早定义了现代代理软件的形态。
* **核心**: Surge 私有核心 (闭源)。
* **优势**:
    * **稳定性**: 极高，不仅是代理工具，更是强大的网络诊断与抓包开发工具。
    * **交互**: 拥有最好的 UI 设计和 Dashboard。
    * **功能**: 强大的脚本功能，Pont (网关模式) 体验极佳。
* **缺点**:
    * **贵**: 价格昂贵且订阅制。
    * **协议保守**: 不支持 VLESS-Reality, Hysteria2, TUIC, AnyTLS 等新兴抗封锁协议（作者认为这些非标准协议不符合其开发美学）。
    * **生态封闭**: 仅适配自家格式。

### Quantumult X (QX) (🇺🇸$7.99)
* **地位**: 性价比之王，曾是 Surge 的最大竞争对手。
* **核心**: QX 私有核心 (闭源)。
* **优势**: 拥有极其强大的自定义分流与重写功能，UI 独特且精美。
* **缺点**:
    * **停更风险**: 作者极其佛系，几乎处于“半退役”状态。
    * **协议缺失**: 不支持 WireGuard, TUIC, VLESS-Reality, Hysteria2 等新协议。

### Loon (🇺🇸$4.99)
* **地位**: 插件玩法的集大成者。
* **核心**: Loon 私有核心 (闭源)。
* **优势**:
    * **易用性**: 插件市场极其丰富，一键安装各种去广告、VIP 破解脚本。
    * **协议**: 比 Surge/QX 开放，支持 WireGuard 等。
* **缺点**: 在极客群体中的网络诊断专业性不如 Surge。

### Egern (🇺🇸$2.99)
* **地位**: 另辟蹊径的极简主义者。
* **核心**: Egern 私有核心 (闭源)。
* **优势**:
    * **极致精简**: 可能是目前 iOS 上最轻量的代理工具。
    * **交互**: 独创的“交互式策略组”，在通知栏即可切换节点。
* **缺点**: 功能相对单一，不适合重度折腾用户。

## 🤖 Android (商业闭源)

### Surfboard (冲浪板)
* **核心**: 闭源核心 (基于 V2ray 修改)。
* **优势**: 可能是安卓端最好看的 UI。
* **缺点**: 协议支持严重阉割（不支持 SS2022, TUIC, VLESS-Reality, Hysteria2）。
