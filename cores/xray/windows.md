# 🪟 Windows - Xray/V2ray Clients

## 🟢 一类 (推荐)

### V2rayN
* **定位**: 自建党 / 单节点梭哈。
* **核心**: 全协议支持 (通过切换 V2fly/Xray/Sing-box/Mihomo 核心 ＋ 各种协议插件)。
* **优势**: 稳定、简单易用。🟢 新协议、新特性迅速跟进，协议覆盖率第一！
* **缺点**: 分流短板、UI 复古，👴🏻 老同志 App。

## 🟡 二类

### Throne
* **说明**: 对原 Nekoray 的 fork。
* **缺点**: 分流短板。

### Vproxy
* **说明**: 三位一体工具，界面美观，但分流短板。

### Nekoray 🔴 (已断更)
* **说明**: UI 优化的 v2rayN，可调用 Singbox/v2ray 核心。

## 🟠 游戏专用

### Netch 🔴 (已断更)
* **专精**: 游戏级 UDP 代理。
* **现状**: 已断更，但除软路由端 PW2 外暂无完美替代品。协议残缺，不适合日常网页。
