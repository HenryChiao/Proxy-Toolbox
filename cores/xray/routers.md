# 🛗 Routers - Xray/V2ray Clients

## 🛜 华硕硬路由

### FancySS (科学上网)
* **核心**: 基于 V2ray/Xray 核心。
* **优势**: 额外整合 Naïve, TUIC 等协议，对 WiFi 5 旧机型友好。
* **缺点**: 分流短板、UI 一般，👴🏻 老同志 App。

### Merlin XrayUI
* **依赖**: Entware 插件。
* **特点**: 勤于更新。

## 🗄 OpenWrt 软路由

### PassWall (PW) (一类)
* **核心**: 任选 Xray/Singbox 等。
* **优势**: 配置弹性大、支持负载均衡、支持搭建节点。
* **缺点**: 分流实现繁琐，👴🏻 老同志 App。

### PassWall2 (PW2) (一类)
* **优势**: 🎮 游戏级 UDP 代理表现、可单独为 UDP 分流。
* **缺点**: 分流操作仍显繁琐。

### ShadowSocksR Plus+ (SSR+) (二类)
* **优势**: 简单易用、支持搭建节点，旧款 CPU 也能轻松运行。
* **缺点**: 分流短板、覆写设置少，👴🏻 老同志 App。

### V2rayA-openwrt (二类)
* **核心**: Xray/V2ray 核心可切换。
* **优势**: Web UI 管理，旧款 CPU 轻松运行。
