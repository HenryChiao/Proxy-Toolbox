# 🤖 Android - Xray/V2ray Clients

## 🟢 一类 (推荐)

### V2rayNG
* **定位**: 自建党 / 单节点梭哈。
* **核心**: 基于 v2ray / xray 核心。
* **优势**: 最早问世的工具之一，成熟稳定、简单易用。
* **缺点**: 分流短板；UI 走经典的 Material Design 公模风；👴🏻 老同志 App。

### Exclave
* **定位**: ～自建党 / 单节点梭哈。
* **核心**: 基于 v2ray 核心。
* **优势**: 稳定、简单易用、链式代理易操作。🟢 新协议、新特性迅速跟进，协议覆盖率《全平台第一》！
* **缺点**: 分流繁琐；UI 公模风。

## 🟡 二类 (美化/变种)

### Surfboard (冲浪板)
* **核心**: 魔改 v2ray 核心的分流。
* **优势**: UI 美观，对 Android 代理工具的审美有极大贡献。
* **缺点**: ✂✂✂ 协议残缺严重 (如 SS2022/TUIC/Vless-Reality/HY2/anyTLS)，覆写可选项少。

### InsightBox (原 v2rayInsight)
* **核心**: 基于 v2ray 和 sing-box 核心的分流。
* **优势**: 简单易用、UI 简洁美观（仿 Surfboard）。

### Xray GUI
* **核心**: 基于 Xray 核心。
* **优势**: 主流协议支持较全。
* **缺点**: 分流短板。

### Vproxy
* **定位**: 自建党 / 单节点梭哈。
* **特点**: 颇有想法的三位一体（VPS 节点搭建 + 探针 + 代理客户端）。
* **缺点**: 分流短板。

## 🟠 三类 / 古早

### ShadowSocks-Android
* **评价**: 当代代理工具的开山鼻祖 (2014.07)。
* **缺点**: ✂✂✂ 严重阉割主流协议，分流短板，👴🏻 老同志 App。
